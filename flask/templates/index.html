Airbnb listing in Beijing

Study Objective

This website is designed to realize interactive data visualization of the Airbnb listings of Beijing. The website provides three parts of analysis, whole city analysis, each district analysis, and get your own advice. The whole city analysis provides an overview of the distribution and detailed information of the whole city. Each district analysis will only show the information and analysis of the selected district. For the last part, get your own advice. K-means clustering method is used to help give you advice based on your own condition. First, enter the number of the clusters you want to try, then enter the acquired information below to retrieve the prices for other listings in the same cluster as yours. Get your own advice shows you a suggested price range of your listing based on the condition.

Data Resources

The data I used in this project is a room listing data from Airbnb official website. See data source here Listings of Airbnb in Beijing. This dataset was collected by Airbnb and all the confidential information was removed from the dataset.

Data Description

There are 72 variables in this dataset. The variables of this dataset can be divided into 5 parts.

Basic information of the listing, including id, url, name, description and so on.

Basic information of the host, including host_id, host_url, host_name, host_location, host_response_time and so on.

Geo information of the listing, including neighbourhood, latitude, longitude and so on.

Booking information of the listing, including room_type, accommodates, amenities, price, availability and so on.

Review information of the listing, including number_of_reviews, number_of_reviews, reviews_per_month.
