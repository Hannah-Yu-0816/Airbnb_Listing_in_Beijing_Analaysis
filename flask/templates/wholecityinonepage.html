Host Type

Define host type:

If the host owns less than 5 listings, define the host type as "Personal". If the host owns less than 20 listings and more than 5 listings, define the host type as "Small scale company". If the host owns less than 100 listings and more than 20 listings, define the host type as "Medium scale company". If the host owns more than 100 listings, define the host type as "Large scale company".

Availability

Define Availability:

If the listing is available for more than 60 days per year, define the availability as "highly available". If the listing is available for less than 60 days per year, define the availability as "lowly available".

Income of the host

Define Income:

Income = The minimum number of days that can be booked * Review per month * Available days per month * Price
